## Install in Claude Desktop / claude.ai

1. Download the skill: [generate-cartoon-video.zip](https://github.com/YOURUSERNAME/generate-cartoon-video/releases/download/v1.0/generate-cartoon-video.zip)
2. Open Claude Desktop (or go to claude.ai)
3. Click your profile icon (bottom left) → **Settings** → **Capabilities**
4. Scroll to **Skills** → click **Upload skill**
5. Select the downloaded ZIP
6. Turn the skill **ON**

Done! Now just type: "Generate cartoon video: A lazy cat becomes a superhero..."
